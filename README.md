## About Me

I am a **Software Engineer based in Dhaka, Bangladesh**, currently working at **Celloscope Limited**.

My work focuses on **full-stack development, REST API design, database-driven systems, microservices, reporting systems, and enterprise workflow automation**.

I regularly work with **Angular, Node.js, PostgreSQL, Oracle, PL/SQL, TypeScript, REST APIs, and Linux-based development environments**.

I am also pursuing an **M.Sc. in Computer Science at Jahangirnagar University**.

Outside of software engineering, I enjoy photography — especially the moon, nature, and the sky.
